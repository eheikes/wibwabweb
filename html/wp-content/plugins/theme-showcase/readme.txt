== Description ==

This plugin will display all themes located in wp-content/themes on a page or post in a showcase gallery with theme screenshots and preview links. 
This is a great plugin for designers looking to showcase their work or anyone that wants an easy way to display and preview multiple themes.

New settings page allows you to choose which themes are displayed

For support visit our Theme Showcase Support Forum:
http://webdevstudios.com/support/forum/wordpress-theme-showcase-plugin/

Features:

    * Display a showcase of your themes
    * Theme screenshots are displayed with titles and descriptions
    * Easily preview all themes loaded in WordPress
    * Allow anyone to preview themes, not just administrators
    * Works with both WordPress 2.x and WordPress MU (2.7 approved!)
    * New settings page lets you pick which themes are displayed

== Installation ==

1. Unzip the file and upload theme-showcase.php to your WordPress plugin folder.
2. Activate the plugin in your WordPress Admin section under the plugins menu
3. Insert this code: [showcase] in your post/page or wp&#95;theme&#95;preview() in your theme and save. That's it!
4. Select which themes you want displayed on the Theme Showcase option page
